# MUI-Form

A collection of Material UI form inputs designed for seamless integration with React Hook Form.

### Supported components:

- [x] Text Field -> `<TextField />`
- [ ] Autocomplete
- [ ] Button
- [ ] Button Group
- [ ] Checkbox
- [ ] Floating Action Button
- [ ] Radio Group
- [ ] Rating
- [ ] Select
- [ ] Slider
- [ ] Switch
- [ ] Transfer List
- [ ] Toggle Button
